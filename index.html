CI/CD & GitOps Deployment on AWS EKS

A fully automated, production-grade DevOps pipeline for a Go web application — from source code commit to live Kubernetes deployment with zero manual steps.

  • Containerized a Go web app with Docker and automated image build and push to DockerHub via GitHub Actions on every commit.
  • Built reusable Helm charts with versioned configs for rolling updates and instant rollbacks on AWS EKS.
  • Implemented GitOps with Argo CD — every chart change in GitHub is auto-synced to Kubernetes. Zero kubectl in production.
  • Configured NGINX Ingress Controller and AWS Load Balancer Controller to expose the app publicly with clean routing.
  • End result: push code → CI builds → image pushed → chart updated → Argo CD deploys → live in Kubernetes.

Biometric Authentication System

A three-factor biometric authentication system combining fingerprint recognition, iris recognition and encrypted credential storage — achieving 95% accuracy with CNN classification and Rubik's Cube Encryption.

  • Three-factor authentication combining fingerprint, iris and encrypted credential storage for layered identity verification.
